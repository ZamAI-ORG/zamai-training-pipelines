# ZamAI Enhanced Tutor Bot 🎓

An intelligent educational assistant powered by **Mistral-7B-Instruct** fine-tuned on Pashto QA datasets.

## Features

- 🧠 **Smart Tutoring**: Context-aware responses using fine-tuned Mistral-7B
- 🗃️ **Dataset Integration**: Leverages Pashto educational QA dataset
- 🌐 **Multilingual**: Supports English, Pashto, and mixed languages
- 📊 **Performance Evaluation**: Built-in model testing and metrics
- 🎯 **Adaptive Learning**: Adjustable difficulty and category focus

## Models & Datasets

- **Model**: [tasal9/ZamAI-Mistral-7B-Pashto](https://huggingface.co/tasal9/ZamAI-Mistral-7B-Pashto)
- **Dataset**: [tasal9/Pashto-Dataset-Creating-Dataset](https://huggingface.co/datasets/tasal9/Pashto-Dataset-Creating-Dataset)

## Usage

1. Select your language preference and difficulty level
2. Ask questions in English or Pashto
3. Get contextual, educational responses
4. Explore dataset examples and run evaluations

Perfect for language learning, cultural education, and Pashto language support!
