# ZamAI Voice Assistant

An advanced multilingual voice assistant powered by:
- 🎤 **Whisper** for speech recognition
- 🧠 **LLaMA-3** for intelligent responses  
- 🔊 **TTS** for natural speech synthesis
- 🌍 **Multilingual** support including Pashto

## Features

- Real-time voice conversation
- Context-aware responses
- Performance analytics
- Multiple language support
- Professional UI with advanced controls

## Usage

1. Click the microphone button to start recording
2. Speak your question or request
3. Get intelligent responses with voice synthesis
4. View conversation history and analytics

## Models Used

- Speech Recognition: `tasal9/ZamAI-Whisper-v3-Pashto`
- Language Model: `tasal9/ZamAI-LIama3-Pashto`
- Text-to-Speech: Integrated TTS pipeline
