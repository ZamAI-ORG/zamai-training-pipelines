# MT5 Multilingual Fine-tuning on Hub Datasets

Interactive Space for fine-tuning mT5 models on your Hugging Face Hub datasets for multilingual tasks.

## Features

- 🌍 Support for 100+ languages including Pashto
- 📚 Browse and select datasets from your Hub account
- 👁️ Preview dataset samples before training
- ⚙️ Configure training for translation or text generation
- 🚀 Launch fine-tuning jobs with LoRA support
- 🧪 Test your models directly in the interface

## Pro Account Features

This Space leverages Hugging Face Pro account capabilities:
- Private dataset access
- Enhanced GPU compute for large models
- Priority job scheduling
- Advanced analytics and monitoring

## Supported Tasks

1. **Translation**: Train bilingual or multilingual translation models
2. **Text Generation**: Multilingual Q&A, summarization, instruction following

## Model Variants

- mt5-small (300M params) - Fast training, good for testing
- mt5-base (580M params) - Balanced performance
- mt5-large (1.2B params) - High quality results
- mt5-xl (3.7B params) - Best quality (requires Pro GPU)

## Language Support

Optimized for Pashto along with support for:
- English, Dari, Urdu, Arabic, Persian
- French, German, Spanish
- And 90+ more languages
