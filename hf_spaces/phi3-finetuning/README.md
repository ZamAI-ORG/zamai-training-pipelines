# Phi-3 Fine-tuning on Hub Datasets

Interactive Space for fine-tuning Phi-3 models on your Hugging Face Hub datasets.

## Features

- 📚 Browse and select datasets from your Hub account
- 👁️ Preview dataset samples before training
- ⚙️ Configure training hyperparameters
- 🚀 Launch fine-tuning jobs with LoRA support
- 📊 Monitor training progress

## Pro Account Features

This Space leverages Hugging Face Pro account capabilities:
- Private dataset access
- Enhanced GPU compute
- Priority job scheduling
- Advanced analytics

## Usage

1. Select a dataset from your Hub account
2. Preview the data to verify format
3. Configure training parameters
4. Launch fine-tuning job
5. Monitor progress and download fine-tuned model

## Model Support

- Phi-3-mini-4k-instruct
- Phi-3-mini-128k-instruct
- Your custom Phi-3 variants
